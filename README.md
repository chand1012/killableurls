# killableurls

A url shortener that you can terminate.

## Requirements
- [Flask](http://flask.pocoo.org/)
- [mysqlclient](https://pypi.python.org/pypi/mysqlclient) for Python 3
- pytz
- A [MySQL](https://www.mysql.com/) database to work with
